startup

function startup(Router,
APIAdapter,
startOptions)

On startup on the server, this function creates an APIAdapter if apiRoutes are present in the startOptions.  Further the actual Barefoot.Router is created an started using Barefoot.Router.Server.start.

Before doing all this, startOptions is checked if an Express.JS app is present (if not, an error gets thrown which will stop the start process).

Parameters

(Barefoot.Router) RouterA Barefoot.Router object.  Important: This is is not an instance of the router, it is the “plain” object which can be used for running “new Router(...)”.
(Object) startOptionsThis object literal is passed to the router when initalizing it.
